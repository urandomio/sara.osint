# Sara's OSINT Resources

A curated collection of OSINT (Open Source Intelligence) tools, techniques, and resources.

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build

# Type checking
bun run check
```

## Tech Stack

- **Astro 5.17.1** - Static site generator
- **Tailwind CSS 4.1** - Utility-first CSS
- **TypeScript** - Type safety
- **Bun** - Fast package manager & runtime

## Deployment

This site is deployed to GitHub Pages via GitHub Actions.
